CoreAdmin Documentation

Tailwind CSS Admin Dashboard Template

Introduction

CoreAdmin is a modern admin dashboard template built with HTML, TailwindCSS, and JavaScript. It provides a responsive layout, analytics dashboard, tables, authentication pages, and multiple UI components for building powerful admin panels.

Installation

1. Download the template package.
2. Extract the ZIP file.
3. Open index.html in your browser.
4. Start customizing the template files.

Customization

You can customize the dashboard by editing the HTML files and modifying TailwindCSS utility classes. All components are built using Tailwind, making it easy to change colors, spacing, typography, and layout structure.
